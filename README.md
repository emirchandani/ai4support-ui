# AI4Support UI

UI-only prototype for an AI-powered support portal with role-based access (User vs. Admin), styled to match the Ai4support theme.

# How to run (view the UI)

1. Clone the repository: git clone <repo-url>
2. Go into the project folder: cd ai4support-ui
3. Install dependencies: npm install
4. Start the app: npm run dev
5. Click local host link to open in browser
